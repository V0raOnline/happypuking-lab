Modelo de Carga Sensorial

Una herramienta explicativa sobre cómo funciona el prefiltrado sensorial y por qué se satura. No es un test, no es un diagnóstico — es para entender y explicar.

Cómo funciona este modelo Marco conceptual y diferencias entre perfiles. Lo dejamos plegado para dar prioridad a la simulación. +

El sistema nervioso recibe miles de estímulos simultáneos: sonidos, luces, texturas, temperaturas y movimiento. Para funcionar necesita un prefiltro: un mecanismo que jerarquiza y atenúa esa información antes de procesarla.

En muchos perfiles neurodivergentes (TEA, TDAH y otros con DPS), la atenuación automática del input opera con menor intensidad basal. No es un procesamiento deficiente: es un desajuste entre el input disponible y la capacidad de filtrarlo. El aire acondicionado, una conversación y la etiqueta de la ropa compiten al mismo nivel de intensidad. El problema no es la persona: es la cantidad de input sin atenuar.

Perfil NT

El prefiltro neurológico atenúa y jerarquiza automáticamente. Los estímulos irrelevantes se suprimen antes de llegar a la consciencia. La carga se mantiene baja en entornos que un perfil DPS viviría como caóticos.

mayor atenuación automática → mismo entorno < input total → menor carga

Perfil DPS / TEA

La atenuación automática del input opera con menor intensidad basal. Más estímulos llegan sin jerarquizar y la carga sube más rápido ante los mismos entornos. Los amortiguadores naturales modulan el umbral disponible; las adaptaciones conductuales reducen el input entrante, pero no modifican la atenuación automática: protegen el sistema.

menor atenuación automática → mismo entorno > input total → mayor carga

Sobre este modelo

Este modelo interactivo simplifica deliberadamente para facilitar la comprensión. No representa todos los perfiles neurodivergentes, no es una herramienta de evaluación y no sustituye ningún tipo de valoración clínica.

Si todo está alto, no implica un fallo personal: implica demasiado input para la capacidad de procesamiento disponible en ese momento. El objetivo del modelo es entender ese desajuste, no medirse con él.

Está pensado también como herramienta de comunicación: para explicar a otras personas cómo funciona este sistema sin tener que justificarse.
