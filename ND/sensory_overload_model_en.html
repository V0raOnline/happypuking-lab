Sensory Load Model

An explanatory tool about how sensory pre-filtering works and why it gets overwhelmed. Not a test, not a diagnosis — it's for understanding and explaining.

How this model works

The nervous system receives thousands of simultaneous stimuli — sounds, lights, textures, temperatures, movement. To function, it needs a pre-filter: a mechanism that prioritises and attenuates that information before processing it.

In many neurodivergent profiles (ASD, ADHD and others with SPD), automatic input attenuation operates at a lower baseline intensity. It is not deficient processing: it is a mismatch between available input and the capacity to filter it. The air conditioning, a conversation and a clothing label compete at the same level of intensity. The problem is not the person — it is the amount of unattenuated input.

NT Profile

The neurological pre-filter automatically attenuates and prioritises. Irrelevant stimuli are suppressed before reaching consciousness. Load stays low in environments that an SPD profile would experience as chaotic.

higher automatic attenuation → same environment < total input → lower load

SPD / ASD Profile

Automatic input attenuation operates at a lower baseline intensity. More stimuli arrive unprioritised and load rises faster in the same environments. Natural buffers modulate the available threshold; behavioural adaptations reduce incoming input — but do not modify automatic attenuation, they protect the system.

lower automatic attenuation → same environment > total input → higher load

About this model

This interactive model deliberately simplifies to aid understanding. It does not represent all neurodivergent profiles, is not an assessment tool, and does not replace any clinical evaluation.

If everything is high, it does not imply personal failure — it implies too much input for the processing capacity available at that moment. The goal of the model is to understand that mismatch, not to measure oneself against it.

It is also designed as a communication tool: to explain to others how this system works without having to justify oneself.
