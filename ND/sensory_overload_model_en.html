Sensory Load Model

An explanatory tool about how sensory pre-filtering works and why it gets overwhelmed. Not a test, not a diagnosis - it's for understanding and explaining.

How This Model Works Conceptual framing and profile differences. Kept folded so the simulation stays front and centre. +

The nervous system receives thousands of simultaneous stimuli: sounds, lights, textures, temperatures, and movement. To function, it needs a pre-filter, a mechanism that prioritises and dampens that information before it is processed.

In many neurodivergent profiles (ASD, ADHD, and others with SPD), automatic input attenuation runs at a lower baseline intensity. This is not defective processing; it is a mismatch between the amount of incoming input and the system's capacity to filter it. Air conditioning, a conversation, and a clothing tag can all compete at the same intensity level. The problem is not the person; it is the amount of unfiltered input.

NT Profile

The neurological pre-filter automatically dampens and prioritises. Irrelevant stimuli are suppressed before they reach conscious awareness. Load stays lower in environments that a person with SPD would experience as chaotic.

stronger automatic attenuation → same environment < total input → lower load

SPD / ASD Profile

Automatic input attenuation runs with lower baseline strength. More stimuli arrive without enough prioritisation, so load rises faster in the same environment. Natural buffers modulate the available threshold; behavioural adaptations reduce incoming input, but they do not change automatic attenuation. They protect the system.

weaker automatic attenuation → same environment > total input → higher load

About this model

This interactive model is deliberately simplified to make the logic easier to understand. It does not represent every neurodivergent profile, it is not an assessment tool, and it does not replace clinical evaluation of any kind.

If everything is high, that does not imply personal failure. It implies too much input for the processing capacity available in that moment. The goal of the model is to understand that mismatch, not to measure yourself against it.

It is also designed as a communication tool, a way to explain to other people how this system works without having to justify yourself.
