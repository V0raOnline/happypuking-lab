Dos pestañas para explorar el análisis morfosintáctico como sistema:

Pestaña Analiza

Guía paso a paso para analizar oraciones morfológica y sintácticamente. Cada paso te indica qué buscar y te da una pista contextual.

  • Selecciona una oración con los botones superiores (N1–N4 indica el nivel de dificultad).
  • Haz clic en las palabras de la oración para seleccionarlas.
  • Pulsa Confirmar selección cuando hayas marcado las palabras del elemento que se pide.
  • ¿Te atascaste? Pulsa ¿Me ayudas? para ver la respuesta y avanzar al siguiente paso.
  • Los elementos identificados quedan marcados con su color de categoría y su etiqueta de función.
  • Usa Cargar mis oraciones para trabajar con oraciones propias en formato YAML.

Pestaña Explora

Mapa visual de relaciones entre categorías morfológicas y funciones sintácticas. Úsalo para entender cómo se conectan las piezas del sistema.

  • Alterna entre Morfología y Sintaxis con el toggle superior.
  • Haz clic en cualquier tarjeta para ver su definición, propiedades, ejemplos y el error frecuente asociado.
  • Las tarjetas relacionadas se iluminan. Haz clic en las etiquetas de relación para saltar entre categorías y funciones.

Formato YAML para oraciones propias

💡 Tip: El formato YAML es ideal para generarlo con un chatbot. Usa este prompt:

Genera X oraciones nuevas para este formato YAML de análisis morfosintáctico. Reglas: - Usa frases totalmente distintas entre sí. - Longitud variable (3–10 palabras). - Cada palabra debe aparecer en tokens. - Todos los tokens deben aparecer en al menos un paso. - Actualiza siempre tokens_correctos, feedback_ok y feedback_error. - Mantén exactamente la estructura YAML. Niveles: 1 = muy simple 2 = simple con CD o CC 3 = estructuras un poco más complejas 4 = subordinadas o estructuras complejas Devuelve solo YAML listo para pegar.

Puedes personalizar la guía de resolución con tus propias frases. Descarga el YAML plantilla desde github.com/V0raOnline/happypuking-lab y edítalo siguiendo esta estructura:

oraciones: - id: 1 # un id único por frase nivel: 1 # nivel de dificultad (1–4) texto: "Mi perro compró un libro." nota: "Instrucciones básicas sobre la frase." trampa: "En qué suelen fallar habitualmente." tokens: # un token por palabra - palabra: "Mi" categoria: determinante subtipo: posesivo # opcional funcion: determinante sintagma: SN-sujeto pasos: # uno por unidad de análisis - id: 1 busca: verbo pista: "¿Qué acción ocurre? Busca la palabra que indica lo que se hace." tokens_correctos: ["compró"] feedback_ok: "Perfecto. 'Compró' es el verbo: núcleo del predicado verbal." feedback_error: "Piensa en qué acción describe la oración. ¿Quién hace qué?"
